# DIO-NET-MVC
 Frontend com .NET

Para criar:
 > dotnet new mvc

Executar:
>dotnet watch run

Os nomes dos metodos tem que bater com o da pagina html!
Rotas:
NomeDaPastaController -> Pasta com nome "NomeDaPasta" -> Metodo com nome "Index" -> Pagina com nome "Index"
